# AutoHome
Install the dependencies.

```sh
$ cd auto_home
$ composer install
$ npm install
```


### For run migrations...


create .env file by .env.example


```sh
$ php artisan migrate
$ php artisan db:seed
```

# Run Application
```sh
$ php artisan serve
```
